# Copy and Paste **Shared Styles**

Copy and Paste shared styles.
It's differs from CMD+ALT+C / V. This plugin copy and paste the **Shared Style**, not just the parameters.

## How to use
1 - Copy the style from one layer

2 - Select target layer(s) and paste :)
